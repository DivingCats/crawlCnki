1. 请求搜索结果的列表页，需要发送两次请求，带cookie
2. 请求下一页时，需要修改referer,下载论文时，一样需要修改referer
3. 爬取失败后，修改UA，继续从失败页码开始新爬取
4. 不要太粗暴，爬取一页，休息一会儿
